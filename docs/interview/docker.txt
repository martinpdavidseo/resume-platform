What is Docker?
Docker packages an application and its dependencies into a portable image that can run consistently across environments.

This is called Docker Layer Caching.
Docker reused Layer 1 and only rebuilt the layer affected by the file change.
Docker caches layers. If a layer hasn't changed, Docker reuses it instead of rebuilding it.
